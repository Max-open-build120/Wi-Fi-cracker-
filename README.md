# What is a Wi-Fi Cracker?

A **Wi-Fi cracker** (also known as a wireless password cracker or WEP/WPA cracker) is a specialized software tool or hardware device designed to test the security of wireless networks by attempting to bypass encryption protocols. These tools are essential for:

- **Penetration Testing**: Authorized security assessments of corporate networks
- **Vulnerability Assessment**: Identifying weak points in wireless infrastructure
- **Cybersecurity Education**: Training security professionals on attack vectors
- **Network Administration**: Auditing personal or business Wi-Fi security

## How Wi-Fi Password Cracking Works

Understanding the technical process behind wireless network attacks helps administrators implement better defenses.

### Step 1: Wireless Packet Capture (Sniffing)

Wi-Fi crackers operate by capturing **802.11 frames** transmitted between wireless clients and access points (APs). This process involves:

- Setting wireless network interface card (NIC) to **monitor mode**
- Capturing encrypted data packets containing authentication handshakes
- Filtering relevant packets for analysis

### Step 2: Exploiting Wi-Fi Encryption Vulnerabilities

| Attack Vector | Description | Target Protocol | Complexity |
|--------------|-------------|----------------|------------|
| **Brute Force Attack** | Systematic password guessing using computational power | WPA/WPA2 | High |
| **Dictionary Attack** | Testing common passwords from pre-compiled wordlists | WPA/WPA2 | Medium |
| **Rainbow Table Attack** | Using pre-computed hash tables for rapid cracking | WPA/WPA2 | Low-Medium |
| **WPS PIN Exploit** | Exploiting 8-digit PIN vulnerability in Wi-Fi Protected Setup | WPS-enabled routers | Low |
| **WEP Cracking** | Statistical analysis of weak RC4 stream cipher | WEP | Very Low |
| **Evil Twin Attack** | Creating rogue access point to capture credentials | All protocols | Medium |
